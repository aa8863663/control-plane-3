Run Benchmark

Trigger a new evaluation run via the API

Benchmark runs are executed via the command line. Use the terminal to run python3 llm_safety_platform.py with the appropriate flags. This page is a reference for run parameters.

Main benchmark run

Run against probes_200.json at all 4 temperatures:

python3 llm_safety_platform.py --data probes_200.json --api <provider> --model <model-id> --temperature 0.0,0.2,0.5,0.8 --out output.csv --runs 1

Control run

Run after every main run — control probes at T=0.0 only:

python3 llm_safety_platform.py --data probes_control_20.json --api <provider> --model <model-id> --temperature 0.0 --out ctrl_output.csv --runs 1
