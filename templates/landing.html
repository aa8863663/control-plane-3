🔬 Key Paper Findings — 11 Models · 8,800+ Evaluations · March 2026

1. Only grok-3-mini passes MTCP. xAI's grok-3-mini achieves 90.5% (grade A) — the only model to clear the 90% threshold. Every other model tested fails to pass, including GPT-4o, the entire Claude family, and Gemini 2.0 Flash.

2. GPT-4o scores D — below GPT-3.5-turbo. OpenAI's flagship model at 65.1% is outperformed by GPT-3.5-turbo (87.9%) and GPT-4o-mini (87.9%). Newer and larger does not mean better on post-correction constraint persistence.

3. Claude shows systematic failure — architecture-level. The entire Claude family clusters 65–69% with less than 2% variance across temperatures. This is not sampling noise — it is a consistent architectural failure mode independent of model tier or cost.

4. Open source beats paid Claude. LLaMA 3.3 70B (75.6%, grade C) outperforms Claude Sonnet 4 (68.1%, grade D) at a fraction of the token cost. NVIDIA Nemotron 70B (88.3%) outperforms every paid model except grok-3-mini.

The MTCP Protocol — Three-Turn Probe Architecture
T1
Initial Probe
Prompt with embedded explicit constraint. Response evaluated against specification.
T2
Correction
Structured correction identifying violation. Recovery latency tracked from this turn.
T3
Reinforced
Final correction with reinforced language. Persistent failure triggers enforcement.
HS
Hard Stop
Automated termination. SHA-256 verified audit trail. Regulator-ready export.
Evaluation Vectors — 200 Probes Across 5 Types
NCA
Negative Constraint
Forbidden words + semantic synonym detection.
SFC
Format Compliance
Valid JSON with required schema fields.
IDL
Density & Length
Exact word count verified deterministically.
CG
Contextual Grounding
Required phrase inclusion from context.
LANG
Multilingual
FR · ES · AR · DE. First multilingual benchmark of this type.

Why MTCP — What No Competitor Measures

Post-Correction Persistence
Every competitor measures if a model follows instructions once. MTCP measures if it stays compliant after being corrected. These are different questions with different answers.
