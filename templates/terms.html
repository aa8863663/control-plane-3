Evaluation requests submitted through the platform are stored securely. Model evaluation results, Evidence Packs, and Release Decision Packs generated by MTCP are provided as tamper-evident JSON artifacts with SHA-256 verification. We do not store model API keys or credentials. Public evidence findings are openly published on this platform for transparency.
