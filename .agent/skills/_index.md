# Skills Index
# Read this first. Load full SKILL.md only when trigger matches.

## audit-runner
Full audit of research estate: database numbers, platform health, paper status.
Triggers: audit, check numbers, verify, what is the status, how many

## evaluation-runner
Runs probe evaluations against provider APIs, verifies database writes, generates Evidence Packs.
Triggers: run evaluations, run probes, evaluate models, benchmark, run overnight

## paper-writer
Drafts extension papers from database findings and framework documentation.
Style matches Paper I (empirical), Paper II (theoretical), Paper III (applied).
Triggers: write paper, draft paper, paper on, extend paper, finish paper

## grant-tracker
Monitors application deadlines, checks status, drafts follow-up emails (does not send).
Triggers: grant, application, follow up, funding status, deadline

## outreach-manager
Tracks contacts, drafts emails and messages (does not send without approval).
Triggers: contact, outreach, email, message, who have I contacted

## content-publisher
Drafts LinkedIn posts, Substack articles, X threads from research findings.
Never posts without explicit approval.
Triggers: post, LinkedIn, Substack, article, write about, comment

## leaderboard-updater
Pulls latest results from database, updates platform leaderboard, flags anomalies.
Triggers: update leaderboard, refresh platform, check leaderboard, leaderboard
